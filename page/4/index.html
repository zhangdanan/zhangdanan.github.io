npm安装报错（operation not permitted, mkdir 'C:\ProgramFiles\nodejs\node_cache\_cacache'）
发表于2024-04-18|error
npm安装报错（operation not permitted, mkdir ‘C:\Program Files\nodejs\node_cache_cacache’）今天安装abbrlink 插件的时候突然报错 npm install EPERM: operation not permitted, mkdir ‘D:\nodejs\node_cache_cacache’ 看了下报错内容，是没权限啊，于是按照网上的教程修改nodejs 文件夹的权限，修改完之后发现还是报这个错误。 又查看了下解决方法，删除c盘用户文件夹下的.npmrc 隐藏文件即可。
