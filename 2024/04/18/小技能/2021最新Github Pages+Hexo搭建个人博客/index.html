  1. 首先要提前安装Git和node.js，如果电脑里没有安装的提前安装，这就不详细说明了。
  2. 要有github账号，没有GitHub账号的去注册一个。
  3. 有git和GitHub账号的，记得提前把ssh密钥添加到GitHub上去，具体的去百度。
